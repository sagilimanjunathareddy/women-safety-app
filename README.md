# women-safety-app
AI-based women safety app with real-time alert system

## 🔥 Features
- 🎤 Scream Detection (AI Model)
- 📍 Live GPS Tracking
- 📸 Automatic Photo Capture
- 📧 Email Alerts
- 📱 SMS & WhatsApp Alerts
- 🗺️ Map Integration
- 📜 Alert History

## 🛠️ Tech Stack
- Python
- Kivy
- TensorFlow Lite
- Twilio API
- Plyer (GPS)

## 🚀 How to Run
pip install -r requirements.txt  
python main.py
